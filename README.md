# 📊 داشبورد بیزینسی تندی

**داشبورد مدیریتی آنلاین** برای لاین‌های کسب‌وکار **تندی**  
(زیرمجموعه هلدینگ خدمات جامع گروه انتخاب)

---

### 🛍️ لاین‌های تحت مدیریت
- **SDA** — لوازم خانگی کوچک
- **Digital** — محصولات دیجیتال
- **Cosmetics & Personal Care** — آرایشی و بهداشتی

---

### ✨ ویژگی‌های فعلی
- ✅ داشبورد کلی + تفکیک هر لاین
- ✅ کارت‌های KPI (فروش کل، تعداد رکورد، بهترین لاین)
- ✅ نمودار فروش ماهانه (Bar Chart)
- ✅ نمودار سهم هر لاین (Pie Chart)
- ✅ جدول فروش با فیلتر لاین
- ✅ تاریخ شمسی + فرمت فارسی اعداد
- ✅ دیتای نمونه (آماده اتصال به دیتای واقعی)

---

### 🛠️ Tech Stack
- HTML + CSS + JavaScript
- Chart.js (نمودارها)
- GitHub Pages (هاست)
