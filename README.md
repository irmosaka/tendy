# Tendy Business Dashboard

داشبورد مدیریتی آنلاین برای لاین‌های کسب‌وکار تندی (زیرمجموعه هلدینگ خدمات جامع انتخاب)

### لاین‌ها
- SDA (لوازم خانگی کوچک)
- Digital
- Cosmetics & Personal Care

### ویژگی‌های فعلی
- داشبورد کلی + تفکیک لاین‌ها
- KPIهای کلیدی (فروش، رشد، حاشیه سود)
- دیتای نمونه (بعداً به دیتای واقعی متصل می‌شود)

Tech Stack: Next.js 15 + TypeScript + Tailwind + Recharts + Supabase
